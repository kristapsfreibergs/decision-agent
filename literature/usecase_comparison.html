Figure 3. Five concrete LLM agentic pipeline configurations across different task types. Left: automated research review following the AI Scientist pattern — open-ended, iterative, code-executing, with HITL as optional quality gate. Right: enterprise decision support — bounded elimination funnel, no web search or code execution, mandatory human approval at both the elimination checkpoint (step 5) and final sign-off (step 9). Purple HITL steps and the red loop arrow highlight the primary structural differences. Both pipelines share the same underlying LLM infrastructure but activate fundamentally different component subsets.
