Fig. 2 Harness components and their corresponding failure modes. RAG (Lewis et al., NeurIPS 2020) is the primary fix for hallucination — grounding each response to retrieved source passages reduces fabrication measurably. Self-RAG (Asai et al., ICLR 2024) extends this with adaptive retrieval, avoiding unnecessary lookups. Self-consistency sampling (Wang et al., ICLR 2023) addresses non-determinism by sampling multiple reasoning paths and majority-voting the answer. Together these components form the minimum viable harness for a production LLM system.

Part II · Research-Backed · 2025

LLMs Are No Longer
Just Transformers

Modern frontier models have absorbed multimodality, tool use, search, code execution, and computer control natively — capabilities that once required external scaffolding are now inside the model itself.

Fig. 3 Architectural evolution from transformer to native multimodal agent. Flamingo (Alayrac et al., NeurIPS 2022) was the first model to natively bridge a frozen language model with a vision encoder — enabling few-shot image-text tasks without external pipelines. GPT-4 (OpenAI, 2023) added native function calling with structured JSON schemas, internalising what ReAct achieved through prompting. GPT-4o (2024) trained natively end-to-end across text, audio, and vision — not stacked encoders but a single omni model. Gemini 1.5 (arXiv:2403.05530) extended the native context to 1M tokens, processing full codebases and hour-long videos in one pass. Claude 3.5 Sonnet's computer use (Anthropic, 2024) scores 14.9% on OSWorld, performing real GUI interactions from screenshots natively.

Fig. 5 The convergence: harness components are being absorbed into the model. In 2020–2022, every capability beyond raw text generation required external scaffolding — RAG, tool calls, chunking, memory, and routing were all harness responsibilities. By 2023, vision and structured function calling moved inside the model (GPT-4, LLaVA). By 2024–2025, search grounding (Gemini), code execution (GPT-4o), computer use (Claude 3.5), and 1M-token native context (Gemini 1.5) mean frontier models handle in one forward pass what previously required a full orchestration stack. What remains external — domain-specific RAG, enterprise memory, custom tool APIs — is the part the model cannot know at training time.

References

Vaswani et al.
NeurIPS 2017 Attention Is All You Need. Introduced the Transformer — multi-head self-attention, no recurrence. Baseline for all subsequent LLMs.
Brown et al.
NeurIPS 2020 Language Models are Few-Shot Learners. GPT-3, 175B. Demonstrated emergent few-shot capabilities but exposed knowledge cutoff and hallucination problems at scale.
Lewis et al.
NeurIPS 2020
arXiv:2005.11401 Retrieval-Augmented Generation for Knowledge-Intensive NLP Tasks. First RAG paper — combines dense retriever with seq2seq generator to ground responses in source documents. Foundation of all modern RAG harnesses.
Wei et al.
NeurIPS 2022
arXiv:2201.11903 Chain-of-Thought Prompting Elicits Reasoning in Large Language Models. Shows multi-step reasoning requires explicit CoT prompting — the bare LLM fails without it.
Alayrac et al.
NeurIPS 2022
arXiv:2204.14198 Flamingo: a Visual Language Model for Few-Shot Learning. First major multimodal LLM bridging frozen language model + Perceiver vision encoder. Handles interleaved text/image sequences natively.
Yao et al.
ICLR 2023
arXiv:2210.03629 ReAct: Synergizing Reasoning and Acting in Language Models. Demonstrates that tool use requires explicit reasoning-action scaffolding — the bare model cannot call APIs without structured prompting.
OpenAI
arXiv 2023
arXiv:2303.08774 GPT-4 Technical Report. Introduced native vision + function calling with structured JSON schemas. First frontier model to internalize tool use as a native output format.
Schick et al.
NeurIPS 2023
arXiv:2302.04761 Toolformer: Language Models Can Teach Themselves to Use Tools. Self-supervised approach — model learns to insert API calls into its own outputs. 2× math performance gain; 98.1% tool invocation recall on LAMA.
Liu et al.
NeurIPS 2023
arXiv:2304.08485 Visual Instruction Tuning (LLaVA). Open-source visual LLM connecting CLIP vision encoder + LLaMA. 85.1% of GPT-4V performance. Established open-source multimodal instruction tuning.
Packer et al.
2023
arXiv:2310.08560 MemGPT: Towards LLMs as Operating Systems. Demonstrates LLMs are stateless by default — persistent memory across sessions requires an external OS-like memory management system.
Li et al.
EMNLP 2023
arXiv:2305.11747 HaluEval: A Large-Scale Hallucination Evaluation Benchmark. Measures 35–60% factual error rate on QA, dialogue, and summarisation. Core empirical evidence that bare LLMs cannot be trusted without grounding.
Liu et al.
TACL 2024
arXiv:2307.03172 Lost in the Middle: How Language Models Use Long Contexts. 20–30% accuracy drop when relevant evidence is in the middle of the context window. U-shaped performance curve across GPT-3.5, GPT-4, LLaMA, Claude.
Google
arXiv 2024
arXiv:2403.05530 Gemini 1.5 Technical Report. 1M-token native context window. Processes 1hr video, 11hrs audio, 30K+ LOC in one forward pass — negating external chunking for many use cases.
Dated Data
arXiv 2024
arXiv:2403.12958 Dated Data: Tracing Knowledge Cutoffs in Large Language Models. Effective cutoffs are earlier than reported due to recency bias in training data. Bare LLMs have unreliable time-awareness without retrieval.
ACL 2025
arXiv:2408.04667 Non-Determinism of "Deterministic" LLM Settings. Accuracy varies up to ±15% at temperature=0 on identical inputs. Performance gap best-to-worst run up to 70%. Raw outputs are not reproducible without harness-level controls.
Anthropic
Oct 2024 Computer Use (Claude 3.5 Sonnet). 14.9% on OSWorld benchmark — previous SOTA was 7.8%. Native GUI interaction from screenshots (mouse, keyboard, browser) without external RPA.
