Explicit pause, external input receipt, and one-shot scheduler resume.

  actor Operator
  actor External as External source
  participant API as server.py API
  participant Runs as runs.service

  Operator->>API: POST /api/runs/{id}/pause
  API->>Runs: pause_run(reason, waiting_for)
  Runs->>Data: append run_paused
  Runs->>Data: append external_input_requested
  Data-->>API: status waiting_external_input

  External->>API: POST /api/runs/{id}/external-input
  API->>Runs: receive_external_input(source_id, input_type, content)
  Runs->>Runs: validate source_id and DSC scope
  Runs->>Data: write external_inputs/{source_id}.json
  Runs->>Data: append external_input_received
  Runs->>Data: append run_resumed
  API->>Exec: execute_ready_workers() in background

  Exec->>Worker: run evaluator when dependencies ready
  Worker->>Tools: list_external_inputs()
  Tools-->>Worker: source list
  Worker->>Tools: read_external_input(source_id)
  Tools-->>Worker: source content
  Worker->>Data: output cites evidence source
  Worker->>Data: PAAP evidence record written after citation
