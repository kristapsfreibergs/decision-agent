## Cloud Infrastructure Providers

### AWS (Amazon Web Services)
- **Headquarters:** Seattle, USA
- **EU Data Residency:** Yes — eu-west-1 (Ireland), eu-central-1 (Frankfurt), eu-north-1 (Stockholm)
- **ISO 27001:** Yes (certificate current)
- **SOC 2 Type II:** Yes
- **GDPR DPA:** Available via AWS Customer Agreement
- **Managed Kubernetes:** EKS (Elastic Kubernetes Service)
- **Managed PostgreSQL:** RDS for PostgreSQL, Aurora PostgreSQL
- **Object Storage:** S3
- **Pricing Tier:** Premium — reserved instances required for cost efficiency
- **Support:** 24/7 enterprise support from EUR 15,000/year
- **Notes:** Largest ecosystem; steepest learning curve; egress costs notable at scale

### Microsoft Azure
- **Headquarters:** Redmond, USA
- **EU Data Residency:** Yes — West Europe (Netherlands), North Europe (Ireland), Germany West Central
- **ISO 27001:** Yes
- **SOC 2 Type II:** Yes
- **GDPR DPA:** Available via Microsoft Product Terms
- **Managed Kubernetes:** AKS (Azure Kubernetes Service)
- **Managed PostgreSQL:** Azure Database for PostgreSQL — Flexible Server
- **Object Storage:** Azure Blob Storage
- **Pricing Tier:** Premium
- **Support:** Unified support from EUR 25,000/year
- **Notes:** Strong Microsoft 365 / Active Directory integration; good for Microsoft-heavy orgs

### Google Cloud Platform (GCP)
- **Headquarters:** Mountain View, USA
- **EU Data Residency:** Yes — europe-west3 (Frankfurt), europe-west4 (Netherlands), europe-north1 (Finland)
- **ISO 27001:** Yes
- **SOC 2 Type II:** Yes
- **GDPR DPA:** Available via Google Cloud DPA
- **Managed Kubernetes:** GKE (Google Kubernetes Engine) — reference implementation of K8s
- **Managed PostgreSQL:** Cloud SQL for PostgreSQL, AlloyDB
- **Object Storage:** Google Cloud Storage
- **Pricing Tier:** Premium (sustained-use discounts automatic)
- **Support:** Enhanced support from EUR 12,000/year
- **Notes:** Strongest Kubernetes offering; ML/AI tooling best-in-class; smaller partner network

### Hetzner Cloud
- **Headquarters:** Gunzenhausen, Germany
- **EU Data Residency:** Yes — Germany (Falkenstein, Nuremberg, Helsinki), Finland
- **ISO 27001:** Yes (ISO/IEC 27001:2017)
- **SOC 2 Type II:** No (not applicable — EU-centric provider)
- **GDPR DPA:** Yes — standard EU DPA, German law
- **Managed Kubernetes:** No managed Kubernetes (use k3s/k0s self-managed or Hetzner dedicated)
- **Managed PostgreSQL:** No managed PostgreSQL (self-managed required)
- **Object Storage:** Hetzner Object Storage (S3-compatible)
- **Pricing Tier:** Budget — 3–5× cheaper than hyperscalers for compute
- **Support:** Community + ticket-based; no enterprise SLA
- **Notes:** Excellent price/performance for EU-only workloads; no managed PaaS; operational burden higher

### OVHcloud
- **Headquarters:** Roubaix, France
- **EU Data Residency:** Yes — France, Germany, UK, Poland
- **ISO 27001:** Yes
- **SOC 2 Type II:** No
- **GDPR DPA:** Yes — French law
- **Managed Kubernetes:** OVHcloud Managed Kubernetes
- **Managed PostgreSQL:** OVHcloud Managed Databases for PostgreSQL
- **Object Storage:** OVH Object Storage (S3-compatible)
- **Pricing Tier:** Mid-range
- **Support:** 24/7 from EUR 1,200/year
- **Notes:** Strong EU sovereignty story; smaller ecosystem than hyperscalers
